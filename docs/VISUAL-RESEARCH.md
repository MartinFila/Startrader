## Patrones visuales recurrentes

### 1. Formato dominante: Video Reels con subtitulos (4 de 5 posts)
- El formato mas exitoso es **video vertical 9:16** con texto superpuesto
- Los videos provienen de: entrevistas del fundador, clips de peliculas/series (Suits), y videos virales redoblados
- Los subtitulos actuan como "gancho" visual aun antes de que el usuario active el audio

### 2. Sistema de color consistente
| Elemento | Color |
|----------|-------|
| Texto principal | **Blanco** (#FFFFFF) |
| Palabras clave / highlight | **Azul medio** (aprox #2E7DD1) |
| Palabras de impacto / enfasis | **Amarillo/dorado** (aprox #FFD700) |
| Fondo de barras/bordes | **Negro puro** (#000000) |
| Barra del handle | **Azul corporativo** con bordes redondeados |

### 3. Tipografia
- **Familia:** Sans-serif geometrica bold (similar a Montserrat Bold o Gotham Bold)
- **Peso:** Siempre Bold o Extra Bold
- **Tamano:** Grande, optimizado para lectura en movil sin zoom
- **Estilo:** Centrado. Algunas palabras en italica para enfasis extra
- **Casing:** Mixed case para subtitulos; ALL CAPS para slides informativos

### 4. Branding permanente
- **Logo EIE:** Tres cuadrados azules con letras blancas "E I E", siempre en esquina superior izquierda
- **Handle:** @el.inversor.eficiente en barra con fondo azul redondeado, siempre centrada en la parte inferior
- **Consistencia:** 100% — todos los posts llevan ambos elementos

### 5. Layout y composicion
- **Videos:** Estructura de sandwich — barra negra con titulo arriba, video en el centro, barra con handle abajo
- **Slides:** Foto de fondo con overlay oscuro, texto en bloque en la mitad inferior
- **Subtitulos:** Siempre centrados, en la mitad inferior del frame, con 2-3 lineas maximo
- **Espacio negativo:** Uso generoso de negro como frame, crea contraste y "premium feel"

### 6. Estrategia de contenido visual
- **Video clips recontextualizados:** Toman clips de series (Suits), noticias internacionales, y videos virales, y los redublan con subtitulos en espanol y un angulo financiero/motivacional
- **Titulos-gancho:** Frases provocativas o curiosas en la parte superior ("Asi es como se ven realmente los coches inteligentes en China")
- **Palabras clave en color:** Las 2-3 palabras mas importantes del texto se resaltan en azul o amarillo, creando un "escaneo rapido" para el ojo

## Recomendaciones para nuestro sistema de templates

### Lo que funciona (replicar)
1. **Subtitulos con highlight de color** — El patron de resaltar 2-3 palabras clave en azul/amarillo es altamente efectivo para engagement en scroll rapido
2. **Branding consistente** — Logo siempre en la misma posicion, handle siempre abajo. Genera reconocimiento de marca inmediato
3. **Fondo negro como frame** — Crea contraste con el contenido y da un look premium
4. **Titulos-gancho arriba** — El titulo provocativo en la parte superior captura atencion antes de que el usuario vea el video
5. **Formato vertical 9:16** — Optimizado para Instagram Reels y Stories

### Lo que podemos mejorar / diferenciar
1. **Variedad de formatos** — EIE depende mucho de video clips. Nosotros podemos crear slides informativos con mayor calidad visual (graficos, datos, infografias)
2. **Paleta mas rica** — Su paleta es funcional pero limitada (azul + amarillo). Podemos usar una paleta mas expresiva manteniendo coherencia
3. **Tipografia mas moderna** — Usar una sans-serif mas contemporanea (Inter, General Sans, Clash Display) para diferenciarnos
4. **Overlays y gradientes** — Su overlay es basico (negro a transparente). Podemos usar gradientes mas sofisticados con nuestros colores de marca
5. **Elementos graficos** — Aniadir iconografia, lineas divisorias, marcos geometricos para slides mas pulidos

### Templates sugeridos a crear
1. **Template "Reel Cover"** — Titulo-gancho arriba (fondo negro), espacio para thumbnail de video en centro, handle abajo
2. **Template "News Slide"** — Foto de fondo + overlay degradado + titulo en bold + fuente/handle abajo
3. **Template "Dato / Estadistica"** — Numero grande destacado + contexto + fuente
4. **Template "Quote / Frase"** — Cita motivacional/financiera con tipografia expresiva
5. **Template "Carrusel educativo"** — Slides con fondo solido oscuro, tipografia limpia, iconografia
